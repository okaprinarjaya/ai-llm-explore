# ai-llm-explore

**IMPORTANT!**: Activate first your preferred Python Environment. Use Python `pyenv` or `venv` to create new or activate the Python Environment.

Required Python 3.12

## Dot env

`.env` file contents

```
USER_AGENT=python-beautifulsoup4

OPENAI_API_KEY=xyz

GOOGLE_API_KEY=xyz
GOOGLE_CLOUD_PROJECT=xyz

LANGCHAIN_TRACING_V2=true
LANGCHAIN_ENDPOINT=https://api.smith.langchain.com
LANGCHAIN_API_KEY=xyz
LANGCHAIN_PROJECT=xyz
```

## Packages installation

Run: `pip install -r requirements.txt` . That will install all required packages needed.
